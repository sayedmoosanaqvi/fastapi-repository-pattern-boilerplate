A professional backend built with **FastAPI** and **SQLAlchemy** using the **Repository Pattern**.

## Key Features
- **CRUD Operations**: Manage clothing items (Shirts, etc.).
- **Clean Architecture**: Decoupled layers (Schemas, Services, Repositories).
- **Automatic Docs**: Interactive API playground via Swagger UI.
- **File Uploads**: Support for image assets.

## Setup
1. Clone the repo: `git clone <your-link>`
2. Create venv: `python -m venv .venv`
3. Install: `pip install -r requirements.txt`
4. Run: `python -m uvicorn app.main:app --reload`
